# Student-Grade-Management-System

Advanced Student Grade Management System using Python-
A Python CLI app to manage student grades with JSON storage, CSV export/import, and performance analysis.

## Features
- Add / Update / Delete / Search / Sort
- Persistent JSON storage
- CSV import/export
- Statistics (Average, Median, Min, Max)
- Optional colored output with Colorama
- Optional pretty tables with Tabulate

## Run
python main.py
